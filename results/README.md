# Results

This folder contains the key evaluation results used for model comparison and selection.

## Structure

- `model_comparison/`: Overall model performance comparisons
- `evaluation/`: Evaluation metrics for each model
- `thresholds/`: Threshold tuning results for the deployed DenseNet121 model
- `per_label/`: Per-label performance metrics for DenseNet121

Only the most relevant results are included to keep the repository clean and focused.
